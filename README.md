# Property Connect — Next.js scaffold

Minimal Next.js 14+ TypeScript starter with Tailwind CSS and ESLint.

Quick start:

1. Install dependencies

   npm install

2. Run dev server

   npm run dev

3. Build for production

   npm run build
   npm start

Notes:
- App Router (app directory) is used.
- Tailwind CSS is configured in styles/globals.css and tailwind.config.js.
- API route placeholder: GET /api/listings
- No demo data included — ready for real models and components.
